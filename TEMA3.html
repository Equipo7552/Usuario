ACCESO AL BIOS PARA CONFIGURAR DISPOSITIVOS Y SECUENCIAS DE ARRANQUE
BIOS

(Basic Input Output Subsystem) Es un software que reside en un chip instalado en la motherboard de la PC. Un método típico para acceder a la pantalla de configuración del BIOS es presionar ESC, F1, F2, F8 o F10 (dependiendo del fabricante del BIOS, pruebe todos si es necesario) inmediatamente después de iniciar su computadora.
La configuración del BIOS permite ejecutar una secuencia de arranque desde un disco duro, una unidad de CD-ROM, la red o un dispositivo externo.
El primer dispositivo de la lista de pedidos tiene prioridad de arranque. Para instalar el software Labdo necesitará asegurarse de que el CD-ROM, el USB o la red (dependiendo del método que utilice para instalar el software) tengan la prioridad de arranque más alta.
Nota: en el caso de una instalación por medio de USB, antes de configurar la prioridad de arranque para el dispositivo, conecte a un puerto USB para garantizar que el BIOS pueda detectarlo.
Pasos para la configuración de la secuencia de arranque de una computadora COMPAQ  500B MT:

    • Reiniciar el computador
    • Inmediatamente presionar F10
    • Pestaña arranque-seleccionar prioridad de dispositivos de inicio
    • Seleccionar 1er dispositivo de inicio
    • Opciones-seleccionar la opción deseada
    • Presionar F10

      Para un conjunto de datos ordenados en forma ascendente o descendente, la mediana es el valor central de los datos.
